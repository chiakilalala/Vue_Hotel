# Hotel_Reservation

## 使用技術

1. Vue Cli3/Vuex/Vue-Router
2. Vue-Hoter-Datepicker


## 改進目標

1.使用Nuxt.js
2.未來可增加金流功能


![](https://i.imgur.com/OqwyFLp.jpg)



## 運行
1. 使用 clone 載到專案資料夾
2. 運行npm install
4. 將根列表 Vue-Hoter-Datepicker/dist 裡的檔案，複製並覆蓋至 node_modules/Vue-Hoter-Datepicker/dist
3. 運行npm run dev開啟本端網頁


## Project setup
```
npm install
```
